# KURIR-COD-SIMULASI-
Game Kurir COD Simulator mengajakmu merasakan serunya menjadi kurir motor tangguh di jalanan Indonesia. Hindari rintangan tak terduga seperti lubang, kucing oren, hingga emak-emak matic. Kumpulkan koin COD untuk meningkatkan upgrade motor sport hedon dan surat RT. Hadapi juga drama mediasi kocak banget saat pelanggan ngamuk menolak bayar paket COD!
